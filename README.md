# 🖋️ RoiTattoo Estudio
### Portafolio Digital de Roimanys Gamez

Web profesional diseñada para exhibir trabajos de tatuaje artístico en **Manatí, Las Tunas**. 
Este sitio sirve como catálogo de alta visibilidad para clientes interesados en realismo y sombreado.

## 🚀 Tecnologías
- HTML5 / CSS3 (Diseño responsivo para móviles)
- GitHub Pages (Hosting)

## 📸 Secciones
- Galería de trabajos destacados (Realismo, Black & Grey).
- Videos de procesos en vivo.
- Enlace directo a WhatsApp y Facebook.
